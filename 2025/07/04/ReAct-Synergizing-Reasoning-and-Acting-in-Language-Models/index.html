ReAct: Synergizing Reasoning and Acting in Language Models

Posted on 2025-07-04 In Agent Valine:

URL

  • paper: https://arxiv.org/pdf/2210.03629
  • code: https://github.com/ysymyth/ReAct

TL;DR

  • дәәзұ»йҖҡиҝҮиҜӯиЁҖжҺЁзҗҶпјҲеҲҶи§Јзӣ®ж ҮгҖҒи°ғж•ҙи®ЎеҲ’пјүдёҺиЎҢеҠЁпјҲиҺ·еҸ–еӨ–йғЁдҝЎжҒҜпјүзҡ„вҖӢвҖӢеҚҸеҗҢжңәеҲ¶вҖӢвҖӢй«ҳж•Ҳе®ҢжҲҗд»»еҠЎпјҲеҰӮзғ№йҘӘж—¶еҠЁжҖҒи°ғж•ҙжӯҘйӘӨпјүгҖӮ
  • еҸ—еҲ°дәәзұ»жҷәиғҪеҗҜеҸ‘пјҢжң¬ж–ҮжҸҗеҮәдёҖз§Қ Reasoning е’Ң Acting зӣёз»“еҗҲзҡ„жЎҶжһ¶пјҢз§°дёә ReActпјҢиҝҷз§Қж–°зҡ„жҺЁзҗҶиҢғејҸеҸҜд»ҘдҪҝиҜӯиЁҖжЁЎеһӢеңЁеӨҚжқӮд»»еҠЎдёӯиЎЁзҺ°жӣҙеҘҪгҖӮ
  • дё»иҰҒдјҳеҠҝеҢ…жӢ¬пјҡ
    • дәӢе®һжҖ§пјҡеҮҸе°‘жЁЎеһӢе№»и§үпјҲиҷҡжһ„дҝЎжҒҜпјүгҖӮ
    • еҶізӯ–йІҒжЈ’жҖ§пјҡеўһејәжЁЎеһӢеңЁеӨҚжқӮзҺҜеўғдёӯзҡ„жіӣеҢ–иғҪеҠӣгҖӮ
    • еҸҜи§ЈйҮҠжҖ§пјҡдәәзұ»еҸҜд»ҘиҝҪиёӘжЁЎеһӢзҡ„жҺЁзҗҶиҪЁиҝ№гҖӮ

Algorithm

ReAct жЎҶжһ¶

  • зәҜжҺЁзҗҶжЁЎеһӢпјҲеҰӮ Chain-of-ThoughtпјүвҖӢвҖӢпјҡжҳ“дә§з”ҹдәӢе®һе№»и§үпјҲеҰӮиҷҡжһ„дҝЎжҒҜпјүе’Ңй”ҷиҜҜдј ж’ӯпјҲеҰӮз®—жңҜжҺЁзҗҶй”ҷиҜҜпјүпјҢзјәд№Ҹе®һж—¶зҺҜеўғдәӨдә’иғҪеҠӣпјҲеӣҫ 1bпјүгҖӮ
  • вҖӢвҖӢзәҜиЎҢеҠЁжЁЎеһӢпјҲеҰӮ WebGPTпјүвҖӢвҖӢпјҡзјәд№Ҹй«ҳеұӮи§„еҲ’иғҪеҠӣпјҢйҡҫд»ҘеӨ„зҗҶеӨҡжӯҘеҶізӯ–пјҲеӣҫ1cпјүгҖӮ
  • ReAct жЁЎеһӢпјҡдәӨжӣҝиҝӣиЎҢжҺЁзҗҶе’ҢиЎҢеҠЁпјҢиғҪеӨҹеңЁеӨҚжқӮд»»еҠЎдёӯиЎЁзҺ°жӣҙеҘҪпјҲеӣҫ 1dпјүгҖӮ

ж ёеҝғиҙЎзҢ®вҖӢвҖӢ

  • йҰ–жҸҗеҚҸеҗҢжЎҶжһ¶вҖӢвҖӢпјҡз»ҹдёҖжҺЁзҗҶдёҺиЎҢеҠЁпјҢи§ЈеҶійқҷжҖҒжҺЁзҗҶдёҺж— и§„еҲ’иЎҢеҠЁзҡ„зјәйҷ·гҖӮ
  • е®һи·өд»·еҖјвҖӢвҖӢпјҡ
    • жҸҗеҚҮжЁЎеһӢвҖӢвҖӢдәӢе®һжҖ§вҖӢвҖӢпјҲеҮҸе°‘е№»и§үпјүдёҺвҖӢвҖӢеҶізӯ–йІҒжЈ’жҖ§вҖӢвҖӢпјҲеӨҚжқӮзҺҜеўғжіӣеҢ–пјүгҖӮ
    • еўһејәвҖӢвҖӢеҸҜи§ЈйҮҠжҖ§вҖӢвҖӢпјҡдәәзұ»еҸҜиҝҪиёӘжҺЁзҗҶиҪЁиҝ№гҖӮ

Thoughts

  • и®әж–ҮжҸҗеҮәзҡ„з®—жі•дёҖзңјејҖй—ЁпјҢйқһеёёз¬ҰеҗҲзӣҙи§үпјҢд»Һе“ІеӯҰи§’еәҰи®ІпјҢдёҖдёӘдәӢзү©зҡ„и¶…йӣҶдёҖиҲ¬йғҪдјҳдәҺе…¶жң¬иә«пјҢеӣ дёәжңҖе·®зҡ„жғ…еҶөе°ұжҳҜи¶…йӣҶйҖҖеҢ–дёәжң¬иә«пјҲеӣ жӯӨеёёи§Ғзҡ„дёҖдёӘжғ…еҶөжҳҜпјҡж–°зҡ„и®әж–Үз§°дёҖзҜҮиҖҒзҡ„и®әж–ҮжҳҜе…¶жҹҗдёӘеҸӮж•°и®ҫзҪ®дёӢзҡ„дёҖдёӘзү№дҫӢпјүгҖӮ
  • иҝҷдәӣжүҖжңүзҡ„ Reasoning е’Ң Acting зӯү Agent еҠҹиғҪпјҢйғҪе»әз«ӢеңЁеӨ§жЁЎеһӢи¶…й•ҝдёҠдёӢж–Үзҡ„еҹәзЎҖдёҠпјҢеӣ жӯӨжЁЎеһӢж”ҜжҢҒи¶…й•ҝдёҠдёӢж–Үй•ҝеәҰжҳҜжЁЎеһӢжҳҜеҗҰе…·жңүй«ҳзә§жҷәиғҪжҪңиҙЁзҡ„е…ҲеҶіжқЎд»¶пјҲиҮіе°‘зҺ°жңүиҢғејҸжҳҜиҝҷж ·пјүгҖӮ
