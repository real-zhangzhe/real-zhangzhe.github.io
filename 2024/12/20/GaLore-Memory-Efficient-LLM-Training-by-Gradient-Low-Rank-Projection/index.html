GaLore: Memory-Efficient LLM Training by Gradient Low-Rank Projection

Posted on 2024-12-20 Edited on 2025-12-11 In SFT Valine:

URL

  • paper: https://arxiv.org/pdf/2403.03507
  • code: https://github.com/jiaweizzhao/GaLore

TL;DR

  • 本文提出一种梯度低秩映射方法，和 LoRA 思想类似，但支持全参数优化，无需冻结模型也无需额外的训练参数，只需在梯度更新时进行低秩映射，从而在训练时节省显存。
  • 由于优化器状态是大模型微调过程中的主要显存消耗，因此本算法本质是在优化器中记录梯度矩阵的低秩近似，从而减少显存消耗。

Algorithm

总体流程

  1. 计算参数的梯度 G∈Rm×nG\in\mathbb R^{m\times n}G∈Rm×n
  2. 计算梯度的低秩近似（通常用 SVD 奇异值分解法）， G≈UVT,U∈Rm×r,V∈Rn×rG\approx UV^T,U\in \mathbb R^{m\times r},V\in \mathbb R^{n\times r}G≈UVT,U∈Rm×r,V∈Rn×r
  3. 记录低秩近似的优化器状态信息，例如：
    1. UUU 和 VVV
    2. UUU 和 VVV 的动量信息
    3. UUU 和 VVV 的二阶动量信息
  4. 将低秩近似的梯度 UVTUV^TUVT 合并变成近似的梯度 G′G'G′，并用 G′G'G′ 更新参数
  5. 由于梯度具有一定程度的稳定性，为了节省奇异值分解带来的损失，UUU 和 VVV 不会在每次迭代中都重新计算，而是每隔 TTT 次迭代更新一次
