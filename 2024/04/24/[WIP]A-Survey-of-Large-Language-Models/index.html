TL;DR

Survey

1. Introduction

  • LLM (large language model) е’Ң PLM (pretrain language model) дё»иҰҒжңүдёүдёӘеҢәеҲ«пјҡ

    1. LLM иЎЁзҺ°еҮәдёҖдәӣд»ӨдәәжғҠи®¶зҡ„ж¶ҢзҺ°иғҪеҠӣпјҢиҝҷдәӣиғҪеҠӣеҸҜиғҪеңЁд»ҘеүҚиҫғе°Ҹзҡ„ PLM дёӯжІЎжңүи§ӮеҜҹеҲ°гҖӮ

    2. LLM е°ҶеҪ»еә•ж”№еҸҳдәәзұ»ејҖеҸ‘е’ҢдҪҝз”Ёдәәе·ҘжҷәиғҪз®—жі•зҡ„ж–№ејҸпјҢдёҺе°ҸеһӢ PLM дёҚеҗҢпјҢи®ҝй—® LLM зҡ„дё»иҰҒж–№жі•жҳҜйҖҡиҝҮжҸҗзӨәжҺҘеҸЈпјҲдҫӢеҰӮ GPT-4 APIпјүгҖӮ

    3. LLM зҡ„еҸ‘еұ•дёҚеҶҚжҳҺзЎ®еҢәеҲҶз ”з©¶е’Ңе·ҘзЁӢгҖӮи®ӯз»ғ LLM йңҖиҰҒеңЁеӨ§и§„жЁЎж•°жҚ®еӨ„зҗҶе’ҢеҲҶеёғејҸе№¶иЎҢи®ӯз»ғж–№йқўе…·жңүдё°еҜҢзҡ„е®һи·өз»ҸйӘҢгҖӮ

  • жң¬ж–Үдё»иҰҒд»ҺеӣӣдёӘж–№йқўд»Ӣз»Қ LLM зҡ„иҝӣеұ•пјҡ

    1. йў„и®ӯз»ғпјҡеҰӮдҪ•и®ӯз»ғеҮәдёҖдёӘжңүиғҪеҠӣзҡ„ LLM

    2. йҖӮй…Қеҫ®и°ғпјҡеҰӮдҪ•д»Һжңүж•ҲжҖ§е’Ңе®үе…ЁжҖ§дёӨдёӘи§’еәҰжңүж•Ҳең°еҫ®и°ғйў„и®ӯз»ғзҡ„ LLM

    3. дҪҝз”ЁпјҡеҰӮдҪ•еҲ©з”Ё LLM и§ЈеҶіеҗ„з§ҚдёӢжёёд»»еҠЎ

    4. иғҪеҠӣиҜ„дј°пјҡеҰӮдҪ•иҜ„дј° LLM зҡ„иғҪеҠӣе’ҢзҺ°жңүзҡ„з»ҸйӘҢжҖ§еҸ‘зҺ°

2. Overview

2.1 еӨ§иҜӯиЁҖжЁЎеһӢзҡ„ж¶ҢзҺ°иғҪеҠӣ

LLM зҡ„ж¶ҢзҺ°иғҪеҠӣпјҲEmergent Abilitiesпјүиў«жӯЈејҸе®ҡд№үдёәпјҡеңЁе°ҸеһӢжЁЎеһӢдёӯдёҚеӯҳеңЁдҪҶеңЁеӨ§еһӢжЁЎеһӢдёӯдә§з”ҹзҡ„иғҪеҠӣпјҢиҝҷйҮҢд»Ӣз»Қдёүз§Қе…ёеһӢж¶ҢзҺ°иғҪеҠӣпјҡ

  1. In-context learningпјҡдёҠдёӢж–ҮеӯҰд№ гҖӮICL иғҪеҠӣжҳҜз”ұ GPT-3 жӯЈејҸеј•е…Ҙзҡ„пјҡеҒҮи®ҫе·Із»ҸдёәиҜӯиЁҖжЁЎеһӢжҸҗдҫӣдәҶдёҖдёӘиҮӘз„¶иҜӯиЁҖжҢҮд»Өе’Ң/жҲ–еҮ дёӘд»»еҠЎжј”зӨәпјҢе®ғеҸҜд»ҘйҖҡиҝҮе®ҢжҲҗиҫ“е…Ҙж–Үжң¬зҡ„еҚ•иҜҚеәҸеҲ—зҡ„ж–№ејҸжқҘдёәжөӢиҜ•е®һдҫӢз”ҹжҲҗйў„жңҹзҡ„иҫ“еҮәпјҢиҖҢж— йңҖйўқеӨ–зҡ„и®ӯз»ғжҲ–жўҜеәҰжӣҙж–°гҖӮ

  2. Instruction followingпјҡжҢҮд»ӨйҒөеҫӘгҖӮйҖҡиҝҮдҪҝз”ЁиҮӘз„¶иҜӯиЁҖжҸҸиҝ°зҡ„ж··еҗҲеӨҡд»»еҠЎж•°жҚ®йӣҶиҝӣиЎҢеҫ®и°ғпјҲз§°дёәжҢҮд»Өеҫ®и°ғпјүпјҢLLM еңЁжңӘи§ҒиҝҮзҡ„д»ҘжҢҮд»ӨеҪўејҸжҸҸиҝ°зҡ„д»»еҠЎдёҠиЎЁзҺ°еҮәиүІгҖӮ

  3. Step-by-step reasoningпјҡйҖҗжӯҘжҺЁзҗҶгҖӮйҖҡиҝҮдҪҝз”ЁжҖқз»ҙй“ҫпјҲChain-of-Thought, CoTпјүжҸҗзӨәзӯ–з•ҘпјҢLLM еҸҜд»ҘйҖҡиҝҮеҲ©з”ЁеҢ…еҗ«дёӯй—ҙжҺЁзҗҶжӯҘйӘӨзҡ„жҸҗзӨәжңәеҲ¶жқҘи§ЈеҶіиҝҷзұ»д»»еҠЎпјҢд»ҺиҖҢеҫ—еҮәжңҖз»Ҳзӯ”жЎҲгҖӮ

2.2 еӨ§иҜӯиЁҖжЁЎеһӢзҡ„е…ій”®жҠҖжңҜ

LLM зҡ„е…ій”®жҠҖжңҜдё»иҰҒеҲҶдёәд»ҘдёӢдә”дёӘж–№йқўпјҡ

  1. ScalingпјҲжү©еұ•пјүпјҡTransformer иҜӯиЁҖжЁЎеһӢеӯҳеңЁжҳҺжҳҫзҡ„жү©еұ•ж•Ҳеә”пјҢжӣҙеӨ§зҡ„жЁЎеһӢ/жӣҙеӨ§зҡ„ж•°жҚ®и§„жЁЎ/жӣҙеӨҡзҡ„и®ӯз»ғи®Ўз®—йҖҡеёёдјҡеҜјиҮҙжЁЎеһӢиғҪеҠӣзҡ„жҸҗеҚҮгҖӮ

  2. TrainingпјҲи®ӯз»ғпјүпјҡеҲҶеёғејҸи®ӯз»ғз®—жі•жҳҜеӯҰд№  LLM зҪ‘з»ңеҸӮж•°жүҖеҝ…йңҖзҡ„пјҢе…¶дёӯйҖҡеёёиҒ”еҗҲдҪҝз”Ёеҗ„з§Қе№¶иЎҢзӯ–з•ҘгҖӮдёәдәҶж”ҜжҢҒеҲҶеёғејҸи®ӯз»ғпјҢе·Із»ҸеҸ‘еёғдәҶдёҖдәӣдјҳеҢ–жЎҶжһ¶жқҘдҝғиҝӣе№¶иЎҢз®—жі•зҡ„е®һзҺ°е’ҢйғЁзҪІпјҢдҫӢеҰӮ DeepSpeed е’Ң Megatron-LMгҖӮжӯӨеӨ–пјҢдјҳеҢ–жҠҖе·§еҜ№дәҺи®ӯз»ғзЁіе®ҡжҖ§е’ҢжЁЎеһӢжҖ§иғҪд№ҹеҫҲйҮҚиҰҒпјҢдҫӢеҰӮйў„и®ӯз»ғд»Ҙе…ӢжңҚи®ӯз»ғжҚҹеӨұжҝҖеўһе’Ңж··еҗҲзІҫеәҰи®ӯз»ғзӯүгҖӮ

  3. Ability elicitingпјҲиғҪеҠӣеј•еҜјпјүпјҡеңЁеӨ§и§„жЁЎиҜӯж–ҷеә“дёҠйў„и®ӯз»ғд№ӢеҗҺпјҢLLM е…·еӨҮдәҶдҪңдёәйҖҡз”Ёд»»еҠЎжұӮи§ЈеҷЁзҡ„жҪңеңЁиғҪеҠӣгҖӮз„¶иҖҢпјҢеҪ“ LLM жү§иЎҢдёҖдәӣзү№е®ҡд»»еҠЎж—¶пјҢиҝҷдәӣиғҪеҠӣеҸҜиғҪдёҚдјҡжҳҫејҸең°еұ•зӨәеҮәжқҘгҖӮдҪңдёәжҠҖжңҜжүӢж®өпјҢи®ҫи®ЎеҗҲйҖӮзҡ„д»»еҠЎжҢҮд»ӨжҲ–е…·дҪ“зҡ„ ICL зӯ–з•ҘеҸҜд»ҘжҝҖеҸ‘иҝҷдәӣиғҪеҠӣгҖӮ

  4. Alignment tuningпјҲеҜ№йҪҗеҫ®и°ғпјүпјҡInstructGPT и®ҫи®ЎдәҶдёҖз§Қжңүж•Ҳзҡ„еҫ®и°ғж–№жі•пјҢдҪҝ LLM иғҪеӨҹжҢүз…§жңҹжңӣзҡ„жҢҮд»ӨиҝӣиЎҢж“ҚдҪңпјҢе…¶дёӯеҲ©з”ЁдәҶ еҹәдәҺдәәзұ»еҸҚйҰҲзҡ„ејәеҢ–еӯҰд№ жҠҖжңҜпјҲRLHFпјүпјҢйҮҮз”ЁзІҫеҝғи®ҫи®Ўзҡ„ж ҮжіЁзӯ–з•ҘпјҢе®ғе°Ҷдәәзұ»еҸҚйҰҲзәіе…Ҙи®ӯз»ғеҫӘзҺҜдёӯгҖӮ

  5. Tools manipulationпјҲж“ҚдҪңе·Ҙе…·пјүпјҡеҲ©з”ЁеӨ–йғЁе·Ҙе…·еҸҜд»ҘиҝӣдёҖжӯҘжү©еұ• LLM зҡ„иғҪеҠӣгҖӮдҫӢеҰӮпјҢLLM еҸҜд»ҘеҲ©з”Ёи®Ўз®—еҷЁиҝӣиЎҢеҮҶзЎ®и®Ўз®—пјҢеҲ©з”Ёжҗңзҙўеј•ж“ҺжЈҖзҙўжңӘзҹҘдҝЎжҒҜпјҢиҝҷз§ҚжңәеҲ¶еҸҜд»Ҙе№ҝжіӣжү©еұ• LLM зҡ„иғҪеҠӣиҢғеӣҙгҖӮ

2.3 GPT зі»еҲ—жЁЎеһӢзҡ„жј”иҝӣ

  1. GPT-1: 2018 е№ҙ

    • 2018 е№ҙпјҢOpenAI еҸ‘еёғдәҶ GPT-1пјҢд»ЈиЎЁз”ҹжҲҗејҸйў„и®ӯз»ғпјҲGenerative Pre-TrainingпјүгҖӮ

    • GPT-1 жҳҜеҹәдәҺз”ҹжҲҗеһӢзҡ„гҖҒд»…еҗ«жңүи§Јз ҒеҷЁзҡ„ Transformer жһ¶жһ„ејҖеҸ‘зҡ„пјҢе№¶йҮҮз”ЁдәҶж— зӣ‘зқЈйў„и®ӯз»ғе’Ңжңүзӣ‘зқЈеҫ®и°ғзҡ„ж··еҗҲж–№жі•гҖӮ

    • GPT-1 дёә GPT зі»еҲ—жЁЎеһӢе»әз«ӢдәҶж ёеҝғжһ¶жһ„пјҢе№¶зЎ®з«ӢдәҶеҜ№иҮӘз„¶иҜӯиЁҖж–Үжң¬иҝӣиЎҢе»әжЁЎзҡ„еҹәжң¬еҺҹеҲҷпјҢеҚійў„жөӢдёӢдёҖдёӘеҚ•иҜҚгҖӮ

  2. GPT-2: 2019 е№ҙ

    • е°ҶеҸӮж•°и§„жЁЎеўһеҠ еҲ°дәҶ 15 дәҝпјҢе№¶дҪҝз”ЁеӨ§и§„жЁЎзҡ„зҪ‘йЎөж•°жҚ®йӣҶ WebText иҝӣиЎҢи®ӯз»ғгҖӮ

    • е®ғж—ЁеңЁйҖҡиҝҮж— зӣ‘зқЈиҜӯиЁҖе»әжЁЎжқҘжү§иЎҢд»»еҠЎпјҢиҖҢж— йңҖдҪҝз”Ёж Үи®°ж•°жҚ®иҝӣиЎҢжҳҫејҸеҫ®и°ғгҖӮ

    • е°Ҫз®Ў GPT-2 ж—ЁеңЁжҲҗдёәдёҖдёӘж— зӣ‘зқЈзҡ„еӨҡд»»еҠЎеӯҰд№ еҷЁпјҢдҪҶдёҺзӣ‘зқЈеҫ®и°ғзҡ„ SOTA ж–№жі•зӣёжҜ”пјҢе…¶ж•ҙдҪ“жҖ§иғҪд»Қз„¶иҫғе·®гҖӮ

  3. GPT-3: 2020 е№ҙ

    • еҸӮж•°и§„жЁЎеўһеҠ еҲ°дәҶ 175 дәҝпјҢеј•е…ҘдәҶ ICL зҡ„жҰӮеҝөпјҢе®ғжҳҜд»Ҙе°Ҹж ·жң¬жҲ–йӣ¶ж ·жң¬зҡ„ж–№ејҸдҪҝз”Ё LLMгҖӮICL еҸҜд»ҘжҢҮеҜј LLM зҗҶи§Јд»ҘиҮӘз„¶иҜӯиЁҖж–Үжң¬зҡ„еҪўејҸз»ҷеҮәзҡ„д»»еҠЎгҖӮ

    • GPT-3 дёҚд»…еңЁеҗ„з§Қ NLP д»»еҠЎдёӯиЎЁзҺ°еҮәиүІпјҢиҖҢдё”еңЁдёҖдәӣйңҖиҰҒжҺЁзҗҶжҲ–йўҶеҹҹйҖӮй…ҚиғҪеҠӣзҡ„зү№ж®Ҡи®ҫи®Ўзҡ„д»»еҠЎдёӯд№ҹиЎЁзҺ°еҮәиүІгҖӮ

    • GPT-3 еҸҜд»Ҙиў«и§Ҷ дёәд»Һ PLM еҲ° LLM иҝӣеҢ–иҝҮзЁӢдёӯзҡ„дёҖдёӘйҮҚиҰҒйҮҢзЁӢзў‘гҖӮе®ғйҖҡиҝҮе®һиҜҒиҜҒжҳҺпјҢе°ҶзҘһз»ҸзҪ‘з»ңжү©еұ•еҲ°еӨ§зҡ„и§„жЁЎеҸҜд»ҘеӨ§е№…еўһеҠ жЁЎеһӢзҡ„иғҪеҠӣгҖӮ

    • OpenAI дёәдәҶжҸҗй«ҳ GPT-3 зҡ„жҖ§иғҪпјҢдҪҝз”ЁдәҶдёӨз§Қзӯ–з•Ҙпјҡ

      1. дҪҝз”Ёд»Јз Ғж•°жҚ®иҝӣиЎҢи®ӯз»ғпјҡ

        • еҺҹе§Ӣзҡ„ GPT-3 жЁЎеһӢпјҲеңЁзәҜж–Үжң¬дёҠиҝӣиЎҢйў„и®ӯз»ғпјүзҡ„дёҖдёӘдё»иҰҒйҷҗеҲ¶еңЁдәҺзјәд№ҸеӨҚжқӮд»»еҠЎзҡ„жҺЁзҗҶиғҪеҠӣпјҢдҫӢеҰӮе®ҢжҲҗд»Јз Ғе’Ңи§ЈеҶіж•°еӯҰй—®йўҳгҖӮ
        • OpenAI еңЁ 2021.07 жҺЁеҮәдәҶ CodexпјҢиҝҷжҳҜдёҖдёӘеңЁеӨ§йҮҸ GitHub д»Јз ҒдёҠеҫ®и°ғзҡ„ GPT жЁЎеһӢпјҢCodex еҸҜд»Ҙи§ЈеҶійқһеёёеӣ°йҡҫзҡ„зј–зЁӢй—®йўҳпјҢе№¶дё”еңЁж•°еӯҰй—®йўҳдёҠжңүжҳҫи‘—зҡ„жҖ§иғҪжҸҗеҚҮгҖӮ
        • е®һйҷ…дёҠпјҢGPT-3.5 жЁЎеһӢжҳҜеңЁеҹәдәҺд»Јз Ғзҡ„ GPT жЁЎеһӢпјҲcode-davinci-002пјүзҡ„еҹәзЎҖдёҠејҖеҸ‘зҡ„гҖӮ

      2. дёҺдәәзұ»еҜ№йҪҗпјҡ

        • InstructGPT еңЁ 2022.01 жҸҗеҮәпјҢд»Ҙж”№иҝӣ GPT-3 жЁЎеһӢдёҺдәәзұ»еҜ№йҪҗиғҪеҠӣпјҢжӯЈејҸе»әз«ӢдәҶдёҖдёӘдёүйҳ¶ж®өзҡ„еҹәдәҺдәәзұ»еҸҚйҰҲзҡ„ејәеҢ–еӯҰд№ пјҲRLHFпјүз®—жі•гҖӮ
        • йҷӨдәҶжҸҗй«ҳжҢҮд»ӨйҒөеҫӘиғҪеҠӣд№ӢеӨ–пјҢRLHF з®—жі•еҜ№дәҺзј“и§Јжңүе®іжҲ–жңүжҜ’еҶ…е®№зҡ„з”ҹжҲҗй—®йўҳеҚҒеҲҶжңүж•ҲпјҢиҝҷеҜ№дәҺ LLM еңЁе®һи·өдёӯзҡ„е®үе…ЁйғЁзҪІиҮіе…ійҮҚиҰҒгҖӮ
        • OpenAI еңЁеҜ№йҪҗз ”з©¶дёӯзҡ„ж–№жі•пјҢжҖ»з»“дәҶдёүдёӘжңүеүҚйҖ”зҡ„ж–№еҗ‘пјҡ
          1. дҪҝз”Ёдәәзұ»еҸҚйҰҲи®ӯз»ғ AI зі»з»ҹ
          2. еҚҸеҠ©дәәзұ»иҜ„дј°
          3. еҒҡеҜ№йҪҗз ”з©¶
