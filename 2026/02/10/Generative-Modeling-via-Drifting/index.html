Generative Modeling via Drifting

Posted on 2026-02-10 In drifting Valine:

URL

  • paper: https://arxiv.org/pdf/2602.04770

TL;DR

  • 本文提出一种单步图像生成的算法 drifting，可以做到一步从噪声得到图片，无需像 diffusion / flow matching 一样多轮迭代得到图片

Algorithm

损失函数设计

L=Eϵ[∥fθ(ϵ)⏟prediction−stopgrad⁡ ⁣(fθ(ϵ)+Vp,qθ ⁣(fθ(ϵ)))⏟frozen target∥2]\mathcal{L} = \mathbb{E}_{\epsilon} \left[ \left\| \underbrace{ f_\theta(\epsilon) }_{\text{prediction}} - \underbrace{ \operatorname{stopgrad}\!\left( f_\theta(\epsilon) + V_{p,q_\theta}\!\left(f_\theta(\epsilon)\right) \right) }_{\text{frozen target}} \right\|^2 \right] L=Eϵ​⎣⎢⎢⎢⎡​∥∥∥∥∥∥∥∥​predictionfθ​(ϵ)​​−frozen targetstopgrad(fθ​(ϵ)+Vp,qθ​​(fθ​(ϵ)))​​∥∥∥∥∥∥∥∥​2⎦⎥⎥⎥⎤​

  • ϵ\epsilonϵ 是随机噪声
  • fθ()f_\theta()fθ​() 是生成模型
  • Vp,qθV_{p,q_\theta}Vp,qθ​​ 是漂移场

漂移场的计算

Vp,q(x)=Vp+(x)⏟正样本吸引−Vq−(x)⏟负样本排斥V_{p,q}(x) = \underbrace{V_p^+(x)}_{正样本吸引} \underbrace{-V_q^-(x)}_{负样本排斥} Vp,q​(x)=正样本吸引Vp+​(x)​​负样本排斥−Vq−​(x)​​

  • ppp 表示真实数据分布
  • qqq 表示模型生成的数据分布

正样本吸引（来自真实数据）

Vp+(x)=1ZpEy+∼p[k(x,y+)⏟标量权重 (y+−x)⏟向量方向]V_p^{+}(x) = \frac{1}{Z_p} \mathbb{E}_{y^{+} \sim p} \left[ \underbrace{k(x, y^{+})}_{标量权重} \, \underbrace{(y^{+} - x)}_{向量方向} \right] Vp+​(x)=Zp​1​Ey+∼p​⎣⎢⎡​标量权重k(x,y+)​​向量方向(y+−x)​​⎦⎥⎤​

负样本排斥（来自生成分布）

Vq−(x)=1ZqEy−∼q[k(x,y−)⏟标量权重 (y−−x)⏟向量方向]V_q^{-}(x) = \frac{1}{Z_q} \mathbb{E}_{y^{-} \sim q} \left[ \underbrace{k(x, y^{-})}_{标量权重} \, \underbrace{(y^{-} - x)}_{向量方向} \right] Vq−​(x)=Zq​1​Ey−∼q​⎣⎢⎡​标量权重k(x,y−)​​向量方向(y−−x)​​⎦⎥⎤​

  • y+y^+y+ 真实样本
  • y−y^-y− 生成样本
  • k(x,y)k(x,y)k(x,y) 相似度核函数，输出标量

相似度核函数

k(x,y)=exp(−∥x−y∥τ)k(x,y)=exp(-\frac{\|x-y\|}{\tau}) k(x,y)=exp(−τ∥x−y∥​)

  • τ\tauτ 表示温度

用一个例子说明损失函数的作用方式

  • 当前生成样本（要算 drift 的点）

x=(0,0)x=(0,0) x=(0,0)

  • 正样本（来自真实分布 ppp）两个点

y1+=(2,0), y2+=(2,2)y_1^+=(2,0),\ y_2^+=(2,2) y1+​=(2,0), y2+​=(2,2)

  • 负样本（来自生成分布 qqq）两个点（通常就是 batch 里的其他生成样本）

y1−=(−1,0), y2−=(0,−1)y_1^-=(-1,0),\ y_2^-=(0,-1) y1−​=(−1,0), y2−​=(0,−1)

Step 1: 算 kernel 权重（谁更“像” x）

  • 正样本的权重
    • 距离：
      • ∥x−y1+∥=2\|x-y_1^+\|=2∥x−y1+​∥=2
      • ∥x−y2+∥=8≈2.83\|x-y_2^+\|=\sqrt 8\approx 2.83∥x−y2+​∥=8​≈2.83
    • kernel:
      • k(x,y1+)=e−2≈0.135k(x,y_1^+)=e^{-2}\approx0.135k(x,y1+​)=e−2≈0.135
      • k(x,y2+)=e−2.83≈0.059k(x,y_2^+)=e^{-2.83}\approx0.059k(x,y2+​)=e−2.83≈0.059
  • 负样本的权重：
    • k(x,y1−)=k(x,y2−)=e−1≈0.368k(x,y_1^-)=k(x,y_2^-)=e^{-1}\approx0.368k(x,y1−​)=k(x,y2−​)=e−1≈0.368

Step 2: 算方向向量（往哪里走）

  • 正样本方向：

y1+−x=(2,0)y2+−x=(2,2)y_1^+-x=(2, 0)\\ y_2^+-x=(2, 2) y1+​−x=(2,0)y2+​−x=(2,2)

  • 负样本方向：

y1−−x=(−1,0)y2−−x=(0,−1)y_1^--x=(-1, 0)\\ y_2^--x=(0, -1) y1−​−x=(−1,0)y2−​−x=(0,−1)

Step 3: 算正向漂移 Vp+V_p^+Vp+​

  • 加权求和

∑k(x,y+)(y+−x)=0.135(2,0)+0.059(2,2)=(0.388,0.118)\sum k(x,y^+)(y^+-x)=0.135(2, 0)+0.059(2, 2)=(0.388, 0.118) ∑k(x,y+)(y+−x)=0.135(2,0)+0.059(2,2)=(0.388,0.118)

  • 归一化

Zp=0.135+0.059=0.194Vp+(x)=10.194(0.388,0.118)≈(2.0,0.61)Z_p=0.135+0.059=0.194\\ V_p^+(x)=\frac{1}{0.194}(0.388, 0.118)\approx (2.0, 0.61) Zp​=0.135+0.059=0.194Vp+​(x)=0.1941​(0.388,0.118)≈(2.0,0.61)

Step 4: 算负向漂移 Vq−V_q^-Vq−​

  • 和 Vp+V_p^+Vp+​ 同理

Vq−(x)=(−0.5,−0.5)V_q^-(x) = (-0.5,-0.5) Vq−​(x)=(−0.5,−0.5)

Step 5: 合并成为最终漂移 Vp,qV_{p,q}Vp,q​

Vp,q(x)=Vp+−Vq−=(2.0,0.61)−(−0.5,−0.5)=(2.5,1.11)V_{p,q}(x)=V_p^+-V_q^-=(2.0, 0.61)-(-0.5,-0.5)=(2.5, 1.11) Vp,q​(x)=Vp+​−Vq−​=(2.0,0.61)−(−0.5,−0.5)=(2.5,1.11)

每个模块的组成

  • Generator: 采用 DiT (Diffusion Transformer) 架构。输入是噪声 + 类别条件。
  • Tokenizer: 使用 Stable Diffusion 的 VAE 将图像压缩到 Latent Space (32×32×432 \times 32 \times 432×32×4)。
  • Conditioning: 使用 AdaLN-Zero (Adaptive Layer Norm) 注入类别信息。这与标准 DiT 一致。
  • Feature Extractor: 用于计算损失函数。论文指出损失是在特征空间中计算的，而非直接在像素空间。用的模型是 MAE 自监督训练得到的。
